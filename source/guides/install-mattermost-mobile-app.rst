Install the mobile app
======================

.. include:: ../_static/badges/allplans-cloud-selfhosted.rst

The Mattermost mobile app is available for `iOS </install/install-ios-app>` and `Android </install/install-android-app>` devices. You can connect the mobile app to both self-hosted and cloud-based Mattermost workspaces.
